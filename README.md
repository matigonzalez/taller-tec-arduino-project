# taller-tec-arduino-project

## Antes instalar:

- node (LTS)

## Como ejecutar:

- git clone https://github.com/matigonzalez/taller-tec-arduino-project.git

- cd taller-tec-arduino-project

- npm i

## Finalmente:

- npm start

- Abrir un navegador y entrar a http://localhost:3000/
